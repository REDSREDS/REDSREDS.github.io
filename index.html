Docker

What is Docker? Why do we need it?

Docker is an engine that runs container, which is used to make it easier to create, deploy and run applications. The container is used to package up all parts of an application like libraries and dependencies.

Container can be regarded as a virtual machine. A container can not see other containers unless specifically configured. It contains everything it needs to run the application.

Image is what creates the container, it is a template for containers. we can create an image or download from the default registry.

Registries stores the images.

Benefits:
For developers, they don’t need to worry about the system that application will run on, they just need to focus on writting codes.
For operationers, docker provides the flexibility where the applicaiton could run on.
It solves dependency conflicts between silimiar applications on the same server, and for multi-server applications, it helps in managing the dependencies on different server.

Merge Sort

It is like a DFS post order traverse based on divide and conquer algorithms. Recursively divide the array (parent node) into two subarrays (two child nodes), and merge them together (the function at each node, is two merge two child nodes together). The merge action will make it a sorted array. And because it is post order traverse, so the subarrays would alrady be sorted, so we just need to merge two sorted arrays into one.

QuickSort And QuickSelect

Quick sort

Quick sort uses the idea of divide and conquer, basically, assign a pivot value, let two pointers moving toward each other, let all values same or less than pivot value on the left, and all values same or greater than pivot value on the right. then respectively deal with the left array and right array.

464. Sort Integers II

Key points:

  1. start >= end : if A is [1, 2], left and right pointer initially were at 0 and 1, but afterward, right will go to -1, and left will go to 1, so we need to include cases when start > end.
  2. left <= right: because if we stop at left == right, then we don’t know which side this index belongs to, or it takes extra steps to figure out, so this is better.

Two Pointers Algorithm

Two pointers algorithm is usually used to deal with problems involving arrays and lists. The pointer usually moves toward one direction and never go back.

Two direcitons

This method always begin with one pointer at the start of the array, the other at the end of the array, those two pointers move towards each other. Problems like palindrome or two sum can be solved using this method.
